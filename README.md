# Sinon.js

This is NoLimit's fork of Sinon.JS, the standalone and test framework agnostic JavaScript test spies, stubs and mocks.

This fork fixes the issues when Sinon is used in conjunction with webpack, with patches from [Andrei Piscus](https://github.com/andrei-picus-hs).

The mentioned patches and repos are avalaible on [uberVU/Sinon.JS](https://github.com/uberVU/Sinon.JS).

## Installation

via NPM (Node Package Manager):

```sh
$ npm install git+https://github.com/nolimitid/Sinon.JS.git
```

currently we have no method of installing this fork via NuGet.

See [CONTRIBUTING.md](CONTRIBUTING.md) for details on how you can contribute to Sinon.JS' main project. Please don't place issues and contribute to this fork.
